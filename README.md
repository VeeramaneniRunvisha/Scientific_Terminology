
# Scientific Terms Project

## Prerequisite
- Python installed on your system.

## One-Time Setup
1.  Open this folder in VS Code.
2.  Open a new terminal.
3.  Navigate to the backend folder:
    ```bash
    cd backend
    ```
4.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## How to Run
1.  **Start the Backend**:
    In the terminal (inside `backend` folder), run:
    ```bash
    uvicorn main:app --reload
    ```
    You should see "Uvicorn running on http://127.0.0.1:8000".

2.  **Start the Frontend**:
    - Open `login.html` or `register.html`.
    - Right-click and select "Open with Live Server" (if you have the extension) OR just double-click the file to open it in a browser.

## Troubleshooting
- If you see "Error connecting to server", make sure the terminal running `uvicorn` is still open and active.
